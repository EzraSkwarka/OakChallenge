How to Use the Row Generator

This page lets you create Oak Tracker progression data without writing JavaScript by hand.

Quick Authoring Rules

  • # Header → section headers
  • Plain text → paragraphs
  • Indented lines → Pokémon lists
  • Enable Use HTML for raw HTML input

Workflow

  1. Fill in Game Data (once)
  2. Fill in Section Metadata
  3. Write the summary
  4. Click Generate Rows
  5. Review the preview
  6. Export all sections
